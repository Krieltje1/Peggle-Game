# Peggle RPG clone!

## Beschrijving
Dit is mijn peggle clone, het moet soort van een rpg worden waar je CP (Cast Points) hebt waarmee je de ballen moet neerschieten (CP geeft je bal bepaalde boosts, als je geen spell cast dan is het een normale), de ballen hebben HP, en ook bepaalde effects bijvoorbeeld 1 bal die jouw bal kapot maakt meteen maar ook zichzelf sloopt! ik wil ook (ooit) een saving & level system toevoegen waarmee je kan opslaan & je spells kan upgraden enzo maar ja word best wel PEAK (CloverPit reference)

oja en als versie 1.0 af is zet ik het ook op itch.io want dan kunnen mensen online/vrienden van mij het downloaden en spelen :)
